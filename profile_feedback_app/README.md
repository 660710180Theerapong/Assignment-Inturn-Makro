คำอธิบาย database

Table ที่มี

Table opinions
id: มี data_type เป็น SERIAL เพื่อสร้าง id อัตโนมัตตามลำดับการสร้าง และเป็น primary key,
name: มี data_type เป็น TEXT เพื่อเก็บข้อมูลชื่อผู้ส่ง โดยไม่จำกัดความยาว จำเป็นต้องกรอก,
email: มี data_type เป็น TEXT เพื่อเก็บข้อมูลอีเมลของผู้ส่ง โดยไม่จำกัดความยาว จำเป็นต้องกรอก,
create_date:มี data_type เป็น DATE เพื่อเก็บข้อมูลวันที่ผู้ส่ง ส่งมา จำเป็นต้องกรอก,
rating: มี data_type เป็น INT เพื่อเก็บข้อมูลคะแนนที่ผู้ส่งให้ เป็นตัวเลข มีค่า default ที่ 1 คะแนน,
opinion: มี data_type เป็น TEXT เพื่อเก็บข้อมูลอีเมลของผู้ส่ง โดยไม่จำกัดความยาว ไม่จำเป็นต้องกรอก
